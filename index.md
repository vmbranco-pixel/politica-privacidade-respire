# Política de Privacidade

O aplicativo RESPIRA respeita a privacidade dos usuários.

## Coleta de Informações
Não coletamos dados pessoais sensíveis. Algumas informações podem ser coletadas automaticamente para melhorar a experiência.

## Uso das Informações
As informações são utilizadas apenas para funcionamento do aplicativo.

## Compartilhamento
Não compartilhamos dados com terceiros.

## Segurança
Adotamos medidas para proteger suas informações.
